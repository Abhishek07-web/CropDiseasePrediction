About This Project

An AI-powered Crop Disease Prediction System built as a final year project at Sandip University

👨‍💻

Developer

Abhishek Gorakh Borade

B.Sc Computer Science (AI, ML & VR)

Sandip University — Batch 2026

🎯

Project Goal

Help farmers detect plant diseases early using AI, reducing crop loss and improving agricultural productivity.

🧠

AI Model

Convolutional Neural Network (CNN) using MobileNetV2 transfer learning trained on the PlantVillage dataset.

📊

Performance

~90%+ Validation Accuracy

~89–92% Test Accuracy

7 disease classes supported

Tech Stack

🐍 Python 3.10+
🌐 Flask
🤖 TensorFlow / Keras
👁️ OpenCV
📊 NumPy / Matplotlib
🎨 HTML / CSS / JS
